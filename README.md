# CubeCube

유전알고리즘을 통해 큐브 솔루션을 도출해보자.

## 설명

유전알고리즘을 큐브 솔루션에 적용해보자. 큐브의 한 회전은 염색체(solution)의 한 유전자에 해당한다.

## 파일 

- `main.cpp`: 기존에 개발했던 3x3x3 큐브의 solution 도출 코드
- `cube_solution_visualizer_sfml.cpp`: AI Generated 큐브 솔루션 시각적으로 보여주는 코드 
- `twotwo.cpp`: 2x2 큐브에 적용한 것. 


## 목표

- SFML 활용 큐브 솔루션 시각적으로 구현하기
- 유전 알고리즘과 기타 알고리즘 활용하여 다양한 방식으로 큐브 Sol 도출하기
